📊 Smart Finance Tracker & Net Worth Dashboard

## 📝 Project Overview
This project is an interactive Finance Dashboard designed to analyze personal income, expenses, savings, and net worth over time. The dashboard provides a complete financial overview by combining multiple perspectives such as spending behavior, saving patterns, and net growth.

The primary goal of this project is to help users understand how they earn, spend, and save money, and whether their financial habits align with their income.

---

## 🎯 Objectives
- Track total income, expenses, and savings
- Analyze spending behavior across different categories
- Understand savings distribution and investment patterns
- Monitor net worth growth over time
- Evaluate financial discipline (spend vs save vs earn)

---

## 📌 Key Features
- 🔹 KPI Cards for Income, Expense %, Savings %, and Net Worth
- 🔹 Category-wise Expense Analysis (Rent, Groceries, EMIs, etc.)
- 🔹 Savings Distribution (Mutual Funds, Emergency Fund, FD, Cash)
- 🔹 Income vs Expense vs Savings Trend Analysis
- 🔹 Net Worth Growth Visualization
- 🔹 Detailed Financial Statement (Year-wise breakdown)
- 🔹 Interactive filters (Year, Month & Year)
